#### 获取一个 D3 对象

方法

+ select 方法选取文档或者一个文档节点中的第一个节点，返回的一个 D3 类型的节点

+ selectAll 方法选取文档或者一个文档节点中的所有节点，最后返回一个包含所有被选择的节点数组

+ append 方法相当于 DOM 操作中的添加节点方法，向目标节点添加一个节点，最后返回添加的这个节点

+ text 设置一个 D3 元素节点的文本, 通常情况下参数为字符串，如果当前的目标元素有通过数据集绑定

则参数可以是一个回调函数，在回调函数里面可以获取到更多的信息，最后返回当前操作的元素。

使用 D3 的一个简单过程就是，

选取要操作的目标节点，通过在这个目标节点上进行相应的操作，最后由 DOM 渲染出结果来。

#### 在 D3 中使用数据

D3 库是一个以数据驱动的可视化库，把一系列的数据通过 D3 中的方法来展示到页面中。

+ data 方法在一个目标 DOM 元素上面应用这些数据，这样 D3 就知道当前的这些数据对应的

目标元素以便后续的各种操作，数据集以参数的形式传到 data 方法中

+ enter 常用的工作模式是为数据集中的每一个数据单元都创建一个元素，enter 方法的目的就是

为数据集中的每一个数据单元都提供一种可以操作的方式，比如 append 添加一个元素。

#### 添加样式

d3 是关于数据的表示和可视化，所有的样式都可以根据数据集来展示。

+ style 给当前的目标元素添加样式，接收两个参数，分别是样式名称和样式的值，样式名称的规则和 DOM 中添加行内样式的规则一致，

样式值通常为一个硬编码的一个值，如果当前的目标元素有通过数据集绑定，则参数可以是一个回调函数，在回调函数里面可以获取到更多的信息，

在回调函数内可以通过数据单元的值来返回样式值。

+ attr 给目标元素添加属性，比如添加一个 class，参数为属性名称和属性值，如果当前的目标元素有通过数据集绑定，

则参数可以是一个回调函数，在回调函数里面可以获取到更多的信息。

#### 创建一个简单的条形图

1 首先为数据集中的每个数据实体创建一个元素

2 给每个元素动态的设置样式

```JavaScript
// bar-chart.js
const dataset = [14, 21, 12, 24, 15, 16]

d3.select('body').selectAll('div')
    .data(dataset)
    .enter()
    .append('div')
    .text(d => d)
    .attr('class', 'bar')
    .style('height', d => d * 10 + 'px')
```

#### D3 中的 SVG 元素

SVG(Scalable Vector Graphics)，全称可缩放矢量图。不管是放大还是缩小都不会存在像素丢失、模糊的情况。

不管是大设备还是小设备都是一样的。在网页中使用 SVG 需要使用 svg 元素。

```JavaScript
// svg-elem.js
const svg = d3.select('body')
                .append('svg')
                .attr('width', 500)
                .attr('height', 100)
```

#### 在 SVG 中添加图形

在 svg 中展示数据，需要在 svg 中添加相应的图形，比如圆形、矩形。

在 svg 中添加一个图形需要指定这个图形的位置和大小，分别是x、y、width、height，

x 为图形距离 svg 区域左边的位置，y 为图形距离 svg 区域上边的距离，width 为图形的宽度，height 为图形的高度。

#### 在 SVG 中使用数据集

在 svg 元素中以数据绑定的方式为每个数据实体添加图形的方式和在文档中添加 div 元素的方式类似。

区别在于在以数据集所有绑定的图形元素必须添加到 svg 元素中，必须要给每一个数据实体图形指定对应的位置和大小。

图形在 svg 元素的默认位置为(0, 0)，位于 svg 区域的左上角，图形的 x 增加则图形会向右边移动，图形的 y 值增加则图形会向下方移动。

这种情况下，y 值的增加使得呈现的图形不太符合通常的图形，所以图形的 y 值的增加应该是图形向上方移动，即视觉上看起来为图形的高度在增加或者减少。

但是方向是向上的。所以图形的 y 值应该是基于 svg 区域的高度和图形本身的高度来动态的计算的。图形 x 的值增加应该注意每个图形之间有一定的空隙。

#### 改变颜色

在 svg 元素中改变一个图形的颜色，通过设置图形的属性 fill 来设置颜色。

#### 添加文本标签

给数据集中的每一个元素添加文本标签的方式和添加图形的方式类似，添加文本标签的元素为 text。

同时也给每一个文本标签指定位置（x 和 y），x 值和图形的 x 值一样，y 值在视觉上看起来应该在图形的上方。

值为图形的 y 值减去一个看起来合适的值，y 值的减少，文本标签就会出现在图形的上方。

#### 添加提示信息

给数据集中的每一个元素添加提示信息通过在每一个图形元素的后面添加一个 title 元素，并通过 text 方法来设置显示的文本。

#### 在 SVG 中添加圆形

添加圆形的方式和添加条形图方式和步骤类似，使用的标签是circle。但是 circle 需要的位置和大小信息和 rect 不太一样。

circle 有三个主要的属性，分别是cx、cy、r，cx 和 cy 为每一个 circle 在 svg 的中心位置，r 为 circle 的大小，

cy 的值需要通过计算来设置，因为其方向的原因。在 circle 图形中，数据集通过是一个嵌套的二维数组。

#### 给圆形添加文本标签

给 circle 添加文本标签的方式和给 rect 添加标签的方式类似，同样也需要给每个标签指定位置，其中 y 的值和 图形的值一样，

x 值要使的标签向右边离图形有一定的距离

通过数据集创建一个线性比例

直接通过数据集来生成图形的方式可能会出现一些无法预料的问题，比如当数据集中存在有数据大于当前 SVG 元素的宽度或者是高度时， 

创建的图形就会溢出 SVG 图形本身。所以，在将数据应用到 SVG 前，应该将数据按照当前 SVG 元素的大小进行处理。

在 D3 中，Scales 可以把原始数据集映射到 SVG 中

+ d3.scaleLinear() 创建一个线性的比例（通常用于线性的比例）,默认情况下，输入的值和输出的值相同

+ scale.domain() 设置数据集中输入范围，参数为一个至少有两个元素的数组, 通常以数据集的范围来设置

+ scale.range() 设置数据集的输出范围，参数为一个至少有两个元素的数组，通常以展示容器的大小来设置

```JavaScript
// scale-demo.js

// create a linear scale
const scale = d3.scaleLinear()

// set the doamin and range
scale.domain([num1, num2])
     .range([num1, num2])

// apply the scale
const scaleNum = scale(originalNum)
```

#### 查找数据集中的最大值和最小值

+ d3.max() 参数为要查找的数据集和一个可选的回调函数，当数据集是一个嵌套的数组的时候，通过回调函数来指定查找的规则，返回数据集中最大的数据实体

+ d3.min() 和 d3.max() 相反

#### 动态比例

对于一个给定的数据集，首先应该设置一个比例的优先级，以便数据的展示符合展示容器的宽度和高度。

添加坐标轴

+ d3.axisBottom 添加 x 轴, 参数为一个基于 x 数据的缩放比例

+ d3.axisLeft 添加 y 轴， 参数为一个基于 y 数据的缩放比例

```JavaScript
// create axes.js

// create data scale
const xScale = d3.scaleLinear()
                        .domain()
                        .range()

// create x axes
const xAxis = d3.axisBottom(xScale)

// draw a axis with g element
svg.append('g')
          .attr('transfrom', 'translate(0, h - padding)')
           .call(xAxis)
```
